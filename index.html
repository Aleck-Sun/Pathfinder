Welcome to the 2D pathfinder visualizer!

To begin the simulation, click Run Pathfinder. Below, there is a legend explaining what the visual is representing. After the simulation has ended, you can click reset to start a new simulation again.

To edit the grid and place new start, end, or wall nodes, select the node you'd like to edit with the drop-down menu on the left. To place start and end nodes, just click anywhere you'd like on the grid. To place a wall node, do the same. To remove wall nodes, simply just click an already existing wall node.

To generate pre-configured walls, use the drop-down menu on the right. Just select the configuration you'd like the simulate and it will generate it for you.

Enjoy!
